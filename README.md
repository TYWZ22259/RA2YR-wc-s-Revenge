# RA2YR-wc-s-Revenge
Red Alert 2 : Yuri's Revenge mod - WC's Revenge
《红色警戒2：尤里的复仇》Mod：WC的复仇 （wcR）

“WC的复仇” 是室长版的后续作品，相较室长版，“WC的复仇”在保持娱乐性的同时尽最大可能进行了平衡。

“WC的复仇”保留了室长版的绝大多数单位与玩法，也从室长版移植了改进一些经典地图。

## 安装
将dist里的文件夹解压到游戏根目录即可
